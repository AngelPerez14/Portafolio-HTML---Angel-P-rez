# Portafolio Personal - Angel Gabriel Pérez Marcano

## Descripción
Portafolio web personal desarrollado con **PHP, MySQL, HTML5 y CSS** que muestra mi información profesional, habilidades técnicas y cuenta con un sistema completo de comentarios. Como estudiante de Ingeniería de Sistemas y desarrollador apasionado, este proyecto demuestra mis habilidades en desarrollo web full-stack.

## Características
- **Diseño Responsive**: Adaptable a dispositivos móviles, tablets y desktop
- **Paleta de Colores**: Azul corporativo y blanco
- **Sistema de Comentarios**: CRUD completo (Crear, Leer, Editar, Eliminar)
- **Base de Datos MySQL**: Almacenamiento seguro de comentarios
- **PHP Backend**: Lógica del servidor y procesamiento de datos

## Tecnologías Utilizadas
- **Frontend**: HTML5, CSS3 
- **Backend**: PHP, MySQL
- **Control de Versiones**: Git
- **Entorno de Desarrollo**: XAMPP (Apache, MySQL, PHP)
