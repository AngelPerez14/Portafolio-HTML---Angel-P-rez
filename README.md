# Portafolio Personal - Angel Gabriel Pérez Marcano

## Descripción
Portafolio web personal desarrollado con HTML5 y CSS que muestra mi información profesional, habilidades técnicas y datos personales como desarrollador y estudiante de Ingeniería de Sistemas.

## Características
- Diseño Responsive: Adaptable a dispositivos móviles, tablets y desktop
- Paleta de Colores: Azul y blanco 
- Sin Dependencias: Código puro HTML + CSS
- SEO Optimizado: Meta tags para mejor posicionamiento
- Accesibilidad: Estructura semántica correcta

## Tecnologías Utilizadas
- HTML5
- CSS (Variables CSS, Flexbox, Grid, Media Queries)
- Git para control de versiones
